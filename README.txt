README MATCH.EXE
================

This is a card matching game written in C++,
using the SFML library.

Needed Libraries: SFML

You can get the version of SFML we used at:
https://www.sfml-dev.org/download/sfml/2.5.1/
Follow the install directions for your system.

To Compile on Command Line, type:
make clean
make match.exe

To Run from Command Line, type:
./match.exe
